# MixedRice

C package for performing the experiments evaluated in Maitra (2013).

This package implements the methodology for the Expectation-Maximization (EM) algorithm in Maitra (2013).

If you use this package, please cite:

R. Maitra. (2013) "On the expectation-maximization algorithm for Rice-Rayleigh mixtures with application to estimating the noise parameter in magnitude MR datasets." *Sankhyā: The Indian Journal of Statistics, Series B,* 75(2):293–318. DOI: 10.1007/s13571-012-0055-y. URL: <https://link.springer.com/article/10.1007/s13571-012-0055-y>

#### Requirements

 R's mathematical library (`Rmath`) and the `LAPACK` package.

To compile, use at the prompt, `make all`

This will create a bunch of executables for the methods evaluated in Maitra (2013).

For details, please see:

R. Maitra. (2013) "On the expectation-maximization algorithm for Rice-Rayleigh mixtures with application to estimating the noise parameter in magnitude MR datasets." *Sankhyā: The Indian Journal of Statistics, Series B,* 75(2):293–318. DOI: 10.1007/s13571-012-0055-y. URL: <https://link.springer.com/article/10.1007/s13571-012-0055-y>
